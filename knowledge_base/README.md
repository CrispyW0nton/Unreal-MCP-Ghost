# Public Knowledge Base

This folder contains reusable MCP and Unreal guidance for agents.

Curated book-derived knowledge files are kept here because they improve agent behavior. Private project notes, asset inventories, generated media, raw PDFs, local-only book paths, and customer or game-specific implementation logs must stay out of git.

Use ignored local folders for private context:

- `knowledge_base/Projects/`
- `knowledge_base/private/`
For book-backed work, keep licensed PDFs on your own machine and store only local paths in `local-book-paths.json`, which is ignored by git.
