# Contributing

Thank you for improving Unreal-MCP-Ghost.

## Branches

- Base stable fixes on `main`.
- Put experimental work on `wip` or a branch created from `wip`.

## Public Repo Rules

- Do not commit private Unreal projects or generated game assets.
- Do not commit raw book PDFs, copied chapters, or local-only reference paths.
- Do not commit credentials, local tunnel URLs, API keys, or personal machine paths.
- Keep examples generic.

## Quality Expectations

- Keep MCP tools deterministic and return structured errors.
- Prefer small, focused changes with tests for server behavior.
- For Unreal plugin changes, verify editor startup and command routing when practical.
