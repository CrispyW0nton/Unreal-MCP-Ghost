# Agent instructions — Unreal-MCP-Ghost

## Unreal MCP plugin and UE bridge

Before editing the Unreal MCP **C++ plugin**, **Python MCP server**, or running **bridge** commands against Unreal Editor:

1. Follow the Cursor project rule **“Unreal MCP + UE5 — book knowledge base”** (`.cursor/rules/unreal-mcp-book-knowledge.mdc`).
2. Read the files it lists under `docs/knowledge-base/` **first**.
3. For deeper detail from the three Packt references, use paths in **`local-book-paths.json`** at the repo root if present (copy from `docs/knowledge-base/local-book-paths.example.json`). That file is gitignored.

**Note:** Restarting Unreal Editor does not automatically load PDF text into the AI. Context comes from this repo’s guides + optional `local-book-paths.json` + your retrieval/RAG setup if you add one later.
